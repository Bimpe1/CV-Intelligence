# CV Intelligence Platform

An intelligent resume analysis platform designed to help recruiters and job seekers quickly evaluate candidate profiles.

## Features

- PDF Resume Upload
- Automated Text Extraction
- Candidate Information Extraction
  - Name
  - Email
  - LinkedIn
  - GitHub
- Technical Skill Detection
- Candidate Scoring System
- Role Recommendation Engine
- Resume Parsing Dashboard

## Tech Stack

### Frontend
- Streamlit

### Backend
- Python

### NLP & Processing
- Regular Expressions
- PDF Parsing

## Example Workflow

Resume Upload
    ↓
Text Extraction
    ↓
Information Extraction
    ↓
Skill Detection
    ↓
Candidate Scoring
    ↓
Role Recommendation
