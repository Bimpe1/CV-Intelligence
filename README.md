<img width="1057" height="821" alt="image" src="https://github.com/user-attachments/assets/e9a828a7-0bc9-4f35-aa49-a69ffee4f11c" /># CV Intelligence Platform

A machine learning-powered resume analysis platform that:

- Extracts text from PDF CVs
- Identifies candidate information
- Extracts technical skills
- Scores candidates
- Recommends suitable technology roles

Tech Stack:
- Python
- Streamlit
- Regex-based NLP
- PDF Processing

Future Enhancements:
- Semantic Job Matching
- Embedding-Based Skill Detection
- LLM-Powered Candidate Summaries
- Recruiter Dashboard
